# Aula 01 - Lógica de Programação

## O que é algoritmo
Algoritmo é uma sequência lógica de passos organizada para resolver um problema.

## Entrada, processamento e saída

Entrada: dados que o sistema recebe.
Processamento: o que é feito com esses dados.
Saída: o resultado final.

## Exemplo do dia a dia

Escovar os dentes:

1. Pegar a escova
2. Colocar pasta de dente
3. Escovar os dentes
4. Enxaguar a boca

## Palavras-chave

Algoritmo: sequência de passos para resolver um problema.

Entrada: informação que entra no sistema.

Processamento: transformação dos dados.

Saída: resultado final.

Lógica: forma organizada de pensar para resolver problemas.
